⑩ 图标库 / 美术素材（替换全站 emoji）

上传一张图片并绑定一个“图标键”，前台对应位置就会用图片替换 emoji（没绑定的位置仍显示 emoji）。常用图标键：nav:home（首页）、nav:items（特勤处）、nav:craft（制作树）、item:xxx（道具名，如 item:5.45x39mm BT）、link:干员图鉴 等。键名可在前台源码里搜 diIcon("键名" 找到。

⑪ 主页内容（视频 / 攻略 / 图片）

在首页一图流顶部展示。视频填 B站/YouTube 链接会自动转成内嵌播放器；图片填图片链接；攻略/资讯填正文。保存后需重新部署静态站才能在前台生效。

⑫ 赞助名单（首页展示）

在首页底部「赞助名单」区块展示。头像填图片链接（可在 ⑤图片库 或 ⑩图标 上传后复制链接粘贴到这里）。保存并重新部署后前台生效。

⑬ 代码维护（全站 docs/、public/ 文件）

代码维护员可在此直接编辑仓库里的网页/样式/脚本文件。所有写入都经 Worker 服务端 GH_TOKEN 提交到 GitHub（Token 绝不下发浏览器）；保存后 GitHub Pages 会自动重新构建（约 1 分钟生效）。worker.js 不在这里改，改 worker 后请在 Cloudflare 控制台重新部署。
