> 更新日期：2026-07-15

## 一、本次重点更新

### 🧩 v47 · 工作台升级为「可视化零代码拼搭器」
- **按成熟零代码平台架构重写**（参考 Bubble / Webflare / Airtable / 普元 等）：**组件库 + 拖拽画布 + 元数据引擎**，用户全程不碰代码。
- **新增 12 个平台组件**：标题、段落、图片、按钮、分割线、间距、列表、卡片网格、表格、表单、数据看板、数据视图。
- **用法**：后台「工作台」→ 进应用/模块 → 新建页面选「🧩 可视化拼搭」→ 点组件库添加 → 配置属性 → 拖拽排序 → 保存即生成页面。
- **双模式兼容**：旧「经典模板」页仍可用、可编辑；新页面走可视化。
- **数据绑定**：数据看板 / 数据视图组件可直接绑定 `data.json` 里的集合，实时出数。

### 🛡 v46 · 安全加固（网站 + 防火墙）
- **修复路径穿越高危漏洞**：Worker 写文件现已收敛到 `docs/` 目录，无法越权写到仓库其它位置。
- **公开提交接口加防护**：`/api/app/submit` 增加 ID 白名单、字段数量/体积上限、单 IP 限流。
- **数据写入校验**：`/api/admin/data` 写入前先校验 JSON，避免整站 `data.json` 被写坏。
- **验证码强随机**：登录验证码改用 Web Crypto，抗爆破提熵。
- **后台看板 XSS 修复**：消除标签跳出脚本的风险。

## 四、部署说明（需你本人执行）

> 按约定 AI 只改本地文件，部署由你来完成。

1. **Worker（防火墙/后端）**
   - 打开 Cloudflare 控制台，编辑 Worker `delta-intel-auth`，粘贴 `cloudflare/worker.js` 全文重新部署。
   - **确认 KV 绑定**（变量名必须一致）：`USERS / SESSIONS / CODES / POSTS / ANNOUNCE / UGC / COMMENTS / ACCESS / PRESENCE / FRIENDS / MESSAGES / AVATARS / WORKBENCH / TEAMS / LB / SECURITY`（未用的可暂不绑，对应功能会优雅提示）。
   - **战队能用起来的关键**：在控制台 `Settings → Bindings` 添加 KV 绑定，变量名 `TEAMS`，指向一个 KV 命名空间（创建 KV 命名空间**不需要实名**，免费账号邮箱即可）。未绑时打开战队页会显示「加载失败：KV 命名空间 TEAMS 未绑定」——这是正常提示，不是 bug。
2. **前端（Pages）**
   - 把 `docs/` 整包重新上传到 GitHub Pages 仓库根目录。
   - 打开网站首页 → **Ctrl+F5** 硬刷，让新版 Service Worker（`delta-intel-v47`）接管并重新预缓存。
