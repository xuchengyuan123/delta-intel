💬 三角洲情报台 · 统一反馈入口

我们想把所有反馈集中到一个地方，避免微信群、QQ、B站私信、邮件到处散落。下面是几种推荐方案，请选择一个你最方便的渠道：

方案 A：腾讯问卷（推荐，零成本）

📋
腾讯问卷收集表
表单数据自动汇总，支持图片/附件，可导出 Excel，国内访问最快。

管理员可在 wj.qq.com 免费创建问卷，把链接填到下面「集中地址」即可。

方案 B：飞书多维表格

📊
飞书多维表格 + 表单视图
可自动推送到群，支持多人协作处理，适合有运营团队的站点。

创建表单视图后，把公开填写链接填到「集中地址」。

方案 C：GitHub Issues

🐙
GitHub Issues 公开反馈
适合技术型站点，可贴截图、打标签、追踪修复进度；但国内访问可能较慢。

需要先把仓库公开，并把 Issues 链接填到「集中地址」。

方案 D：企业微信群 / 微信群

💬
微信群 / 企业微信群收集
最接地气，玩家可直接聊天；但反馈难以归档和检索，需要人工整理。

把群二维码或邀请链接填到「集中地址」。

当前集中地址（管理员配置）

管理员可在后台「网站反馈」面板配置最终跳转地址。当前默认展示本页，等管理员选定方案后统一替换。

如果你有紧急反馈，也可以直接发邮件到：feedback@delta-intel.local（请管理员在后台替换为真实邮箱）。
