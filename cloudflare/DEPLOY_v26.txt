【三角洲情报台 v26 部署说明】
================================================================
本版只改前端（docs/），后端 Worker 无需改动：
  - 图鉴(tujian.html) 与 改枪码(guns.html) 的数据都走「前端直连公开 API + data.json」，
    不新增任何后端接口，所以 **不需要重新 wrangler deploy**。
  - 后台「图鉴管理」面板复用已有的 data.json 写入通道（putDataObj），Worker 原样可用。
  - 只需：上传 docs/ 整包覆盖 → 浏览器 Ctrl+F5 强刷。

部署顺序：先上传前端，再强刷（无需先动 Worker）。

----------------------------------------------------------------
一、改了什么
----------------------------------------------------------------
1) 后台侧边栏「分组可折叠」+「最近使用」（解决“太乱”）
   - 分组标题可点击折叠 / 展开；默认仅展开「总览与账号」，其余收起。
   - 侧边栏顶部新增「最近使用」快捷区：自动记录你最近点过的 6 个功能，下次进来一眼直达。
   - 搜索框保持“只创建一次 + 输入法守卫”，中文打字不再重复。

2) 新增「三角洲图鉴」页 tujian.html（对标 shushu.fan 的图鉴）
   - 数据来源：WaterEve/DeltaForceData 公开 API（免密钥、浏览器直连、CORS 可用，大陆可访问）。
   - 左侧分类（收集品 / 钥匙 / 消耗品 / 曼德尔砖 / 枪械 / 护甲 / 背包 / 胸挂 / 头盔 / 配件），
     右侧网格卡片（缩略图 + 名称 + 稀有度配色徽标）；点卡片弹详情（大图 + 描述 + 结构化属性）。
   - 顶部搜索框按“物品名 / 分类”实时过滤；“全部”一次拉全部分类，“★精选”展示管理员精选。
   - 图片直接用官方资源地址（playerhub.df.qq.com/.../{objectID}.png），本站只做聚合展示，不存游戏资源。

3) 新增「改枪码」页 guns.html（对标 shushu.fan/guns）
   - 卡片化展示改枪方案：枪图、方案名、武器、作者、价格、标签，一键「复制方案」（复制完整配件清单）。
   - 枪图：自动用枪械 API 把武器名映射到真实枪图；映射不到的（非步枪类）显示占位图标，
     也可在后台「改枪方案」面板给单条补 image 字段。
   - 顶部搜索 + 标签筛选。

4) 后台新增「图鉴管理员」角色 + 「图鉴管理」面板
   - 角色：图鉴管理员（仅能进「图鉴管理」），总管理员照旧拥有全部面板。
   - 面板（🗂 图鉴管理）：开关图鉴分类、用数字调整顺序、添加 / 删除分类、
     填“精选物品”JSON（[{"objectID":...,"note":...}]）、改数据源根地址。保存即生效。
   - 这样“多加几个管理员按功能分类”落到了图鉴这一块；后续要再细分（如单独的收集品管理员）
     可在 ROLES / PANELS / PANEL_GROUPS 里照葫芦画瓢加。

5) 首页预警（地震 / 台风）：当前已是右下角「悬浮按钮 + 弹窗」形态（上轮完成），
   本次随版本号一起刷新缓存，确保新用户强刷后拿到最新。

----------------------------------------------------------------
二、你需要做的部署（2 步）
----------------------------------------------------------------
1. 上传前端：把 三角洲情报台-上传包\docs\ 整文件夹覆盖到
   GitHub 仓库 xuchengyuan123/delta-intel 的 docs/（可直接拖拽，或 git push）。
   ⚠ 不要动 cloudflare/、docs/ 里的源文件(ingest.js 等) 与 cloudflare/worker.js —— 本次不更新它们。

2. 浏览器 Ctrl+F5 强刷（清掉旧 Service Worker 缓存；版本已 ?v=25 → ?v=26，
   sw.js 缓存名 delta-intel-v25 → delta-intel-v26）。

（后端 Worker 本次无需 wrangler deploy；若你顺手 redeploy 也无害。）

----------------------------------------------------------------
三、自检（部署后）
----------------------------------------------------------------
- 后台侧边栏：点分组标题能折叠 / 展开；默认只展开「总览与账号」；用过的功能出现在「最近使用」。
- 后台左侧：总管理员能看到「🗂 图鉴管理」；在「分管理员管理」里可新建「图鉴管理员」角色。
- 访问 tujian.html：左侧分类可点，右侧网格出图，点卡片弹详情大图；顶部搜索可用。
- 访问 guns.html：卡片网格显示改枪方案，点「复制方案」能把配件清单复制到剪贴板。
- 首页顶栏 / 左侧菜单：出现「🗂 图鉴」「🔧 改枪码」两个入口。
- 后台「网站更新速览」：出现 v26 条目。
