================================================================
三角洲情报台 · v29 部署说明
================================================================
本版核心：多端设备管理 + 全站手机自适应 + 特勤处实时物价
改动涉及【后端 Worker】与【前端整包】，两步都要做。

----------------------------------------------------------------
一、本版改了什么
----------------------------------------------------------------
【后端 worker.js（必须重新部署）】
  ▸ 多端/设备管理：
    - 登录（总管理员 / 分管理员 / 普通用户）时记录设备信息
      （浏览器、系统、登录时间，由 User-Agent 解析），并维护
      每用户会话索引 sessions:<email>（复用现有 SESSIONS KV，
      无需新增 KV 绑定）。
    - 新增接口：
        GET  /api/sessions        → 列出当前账号的全部登录设备
                                     （标记「本机」），需登录
        POST /api/sessions/revoke → 踢下线指定设备（只能踢自己
                                     账号下的设备）
    - 退出登录 / 踢人 同步清理会话索引。
  ▸ 说明：后端本来就支持「电脑 + 手机同时登录」（每次登录独立
    token，互不踢线），本版只是把这件事显性化并加上「设备管理」面板。

【前端 docs/（整包覆盖上传）】
  ▸ admin.html：新增「设备管理」面板（侧栏「总览与账号」组，所有
    后台账号均可见）；后台布局改为「抽屉式」自适应（手机汉堡菜单
    开合 + 遮罩），不再是窄屏直接隐藏侧边栏导致无法操作。
  ▸ js/crafting.js：特勤处产物接入免费实时物价 API
    （caiweilv/DeltaForcePrice，游戏内交易行真实成交价），自动用
    实时售价重算「实时利润 / 小时利润」，取不到回退静态并标注。
  ▸ css/style.css：新增横竖屏 / 小屏自适应增强（表格横滑、输入框
    ≥16px 防 iOS 缩放、图鉴/卡战备/音乐/改枪码 移动端断点）。
  ▸ sw.js：CACHE 名 → delta-intel-v29。
  ▸ 全站静态资源版本号 ?v=28 → ?v=29（强制浏览器刷新缓存）。

----------------------------------------------------------------
二、KV / 环境变量（重要）
----------------------------------------------------------------
  ▸ SESSIONS KV：已存在（登录态存放处）。设备管理复用它、自动维护
    索引，**无需新增 KV 绑定**。
  ▸ LB KV：v28 的「实时排行榜」用到，若上一版没绑，本版排行榜仍
    需绑定（变量名 LB）。不绑则排行榜显示「未启用」，不影响其他。
  ▸ ORZICE_TOKEN / EmailJS 等：与 v27/v28 一致，按需配置。

----------------------------------------------------------------
三、部署步骤（3 步）
----------------------------------------------------------------
1) 部署后端 Worker（控制台或 wrangler）：
   - 控制台：Workers → delta-intel-auth → 把 cloudflare/worker.js
     全文粘贴覆盖 → 保存并部署。
   - 或本地：wrangler deploy（需已登录、wrangler.toml 指向该 Worker）。
   - 确认环境变量/Secrets 仍在（GH_TOKEN、REPO、SESSIONS 绑定、
     LB 绑定等）。

2) 上传前端：把桌面「三角洲情报台-上传包\docs\」整个文件夹覆盖
   上传到 GitHub 仓库 xuchengyuan123/delta-intel 的 docs/ 目录
   （含改动的 admin.html / crafting.js / style.css / sw.js 及所有
   ?v=29 引用的 html/js）。

3) 浏览器打开网站 → Ctrl+F5 强制刷新（清掉旧 Service Worker 缓存）。

----------------------------------------------------------------
四、自检清单（部署后）]
----------------------------------------------------------------
  □ 用电脑登录后台 → 左侧栏「总览与账号」出现「📱 设备管理」。
  □ 用手机浏览器登录同一账号 → 后台可正常打开（点左上角 ☰ 展开
    侧边栏，点功能后自动收起）。
  □ 「设备管理」里能看到「本机」与「其他设备」；点「踢下线」后该
    设备下次请求会被要求重新登录。
  □ 前台「特勤处产物」页顶部显示「实时」角标与实时物价更新时间；
    断网/源不可用时显示「静态」且不编造数字。
  □ 图鉴 / 卡战备 / 音乐 / 改枪码 在手机竖屏、横屏下布局正常，
    表格可横向滑动。
  □ 控制台 Network 无 worker.js 报错；/api/sessions 返回当前设备列表。
