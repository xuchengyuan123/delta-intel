【三角洲情报台 v25 部署说明】
================================================================
本版全是前端改动，没有改后端 Worker，所以**不需要重新 wrangler deploy**。
只要：把 docs/ 整包覆盖上传到 GitHub 仓库，然后 Ctrl+F5 强刷即可。

----------------------------------------------------------------
一、改了什么
----------------------------------------------------------------
1) 修复「后台侧边栏搜索框输三个字母」的 bug（最影响体验）
   - 根因：renderSide() 之前每次敲键盘都把整个侧边栏（含搜索框）销毁重建，
     中文输入法组字过程中输入框被反复销毁/重建，组字串被重复提交 → 敲一字出三字。
   - 修法：搜索框只在首次创建一次，之后只重渲染「功能列表」；input 监听加
     e.isComposing 守卫（组字中不打断），compositionend 时再刷新。
   - 影响：后台所有需要打字的地方（搜索、验证码、各种编辑框）都跟着稳了。

2) 邮箱验证码「整一下」
   - 后端逻辑（sendCode / 验证 / dev 模式回显）本身是对的。
   - 真正卡你的是上面的输入框 bug —— 之前验证码/邮箱框会重渲染，导致字输不进去。
     现在 bug 修掉，开发模式下「获取验证码」后页面会把验证码直接显示在下方，复制填进去即可。
   - 说明：要「真发邮件」需要自有域名（你目前无法实名注册），所以走开发模式回显是现实可用路径。
     若以后想真发信，可在 Cloudflare 配 Email Routing + MailChannels，或用 EmailJS（免费、免域名，需你注册拿 3 个 key）。

3) 新增「卡战备 / 配装推荐」页（参考 shushu.fan/kzb）
   - 入口：首页顶栏「🎴 卡战备」+ 左侧菜单「网站相关 → 卡战备 / 配装」+ 直接访问 kzb.html。
   - 内容：按地图列卡战备掉率（机密 / 绝密，单位 W）；枪械优先 / 均衡套装 / 胸挂优先
     三套配装推荐，含售价 / 战备 / 假账与每件物品价格 + 涨跌幅（绿涨红跌）。
   - 数据可改：后台「代码维护」面板编辑 data.json 的 kzb 字段（maps / loadouts）即可，
     已用抓取到的内容做了一版种子数据，你可按最新行情更新。

4) 新增首页「天气」小组件
   - 位置：首页内容区顶部（顶栏也有「🌤 天气」入口跳回首页）。
   - 数据源：Open-Meteo（免密钥、浏览器直连、支持 CORS，大陆可访问）。
   - 默认上海；框里输入城市名（如 北京 / 广州）回车即可切换，选择会自动记忆。
   - 显示：温度、天气图标、湿度、风速。

5)【本版新增】首页「地震 / 台风预警」卡片（天气卡片下方）
   - 自动匹配地区：复用你天气里设的城市（存在 localStorage di_weather_city）作为关注地区，
     无需重复设置；没设过则默认上海。
   - 地震：调用中国地震台网(CENC，经 Wolfx 代理)数据，显示震源位置、经纬度、深度、发生时间，
     并算出「距你 X km」；1000km 内高亮 ⚠ 提示（"可能影响你所在地区"）。
   - 台风：调用中央气象台台风网(nmc.cn)数据，显示活跃台风的实时中心位置、最大风速、气压、
     移动方向(实时风向，如 西北/北西北)与移速，2000km 内高亮 ⚠ 提示。
   - 数据每 5 分钟自动刷新；两个来源 API 均免密钥、浏览器直连、支持 CORS，大陆可直接访问，
     纯前端实现，**没有改后端 Worker**。

----------------------------------------------------------------
二、你需要做的部署（2 步）
----------------------------------------------------------------
1. 上传前端：把 三角洲情报台-上传包\docs\ 整文件夹覆盖到
   GitHub 仓库 xuchengyuan123/delta-intel 的 docs/（可直接拖拽，或 git push）。
2. Ctrl+F5 强刷（清掉旧 Service Worker 缓存，版本已升到 v25）。

（本版无需 npx wrangler deploy；若你顺手也 redeploy 一次也不会出错。）

----------------------------------------------------------------
三、自检（部署后）可浏览器控制台/网络里看
----------------------------------------------------------------
- 后台侧边栏搜索框：用中文输入法打字，敲一字只出一字（不再三字）。
- 访问 kzb.html：能看到地图卡战备掉率 + 三套配装。
- 首页顶部：天气卡片显示当前城市温度；改城市后能切换并记忆。
- 首页天气卡片下方：出现「地震 / 台风预警」卡片，地震列表有震源+距你距离，
  台风列表（若有活跃台风）显示移动方向与中心位置；切换天气城市后预警地区随之更新。
- 后台「网站更新速览」：出现 v25 条目。
